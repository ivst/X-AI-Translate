Privacy Policy AI Translate for X and YouTube (X-AI-Translate)

Summary
- The extension processes text you select or tweet text you ask to translate.
- Text is sent to the API endpoint you configure (OpenAI-compatible).
- The extension stores your API settings in Chrome storage to keep them between sessions.

Data We Collect
- Selected text or tweet text you explicitly request to translate.
- API endpoint and model settings you configure.

How We Use Data
- To send translation requests to the provider you choose.

Data Storage
- Settings are stored in Chrome storage (sync).
- The last translation or error may be stored locally (local storage) for fallback display.

Data Sharing
- Translation text is sent only to the API endpoint you configured.
- We do not sell or share data with third parties beyond the configured provider.

Retention
- Settings remain until you change or remove them.
- The last translation/error remains until cleared.
