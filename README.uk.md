# AI Translate для X і YouTube

</p>

Розширення Chrome для перекладу вибраного тексту, дописів X (Twitter) і коментарів YouTube за допомогою власного ключа API (сумісні з OpenAI і підтримувані нативні постачальники).

## особливості
- Перекладіть виділений текст за допомогою контекстного меню або гарячої клавіші `Alt+Shift+T` (потоковий вихід).
- Кнопка вбудованого перекладу під X дописами/відповідями та коментарями YouTube (потоковий вихід).
- Додаткова кнопка швидкого перекладу біля виділеного тексту.
- Підтримує OpenAI, Claude, Gemini, DeepSeek, YandexGPT, OpenRouter і власні кінцеві точки, сумісні з OpenAI.
- Список моделей OpenRouter із безкоштовним фільтром і кеш-пам’яттю.
- Автоматичне завантаження списків моделей для OpenAI, Claude, Gemini, OpenRouter і YandexGPT.
- Режими виводу: тост у нижньому правому куті або модальний по центру (без розширень спливаючого виводу).
- Вибір цільової та вихідної мови (доступне автоматичне визначення).
- Ключі API можна зберігати локально (за замовчуванням) або синхронізувати між пристроями (необов’язково, менш безпечно).
- Результати збережено в «Останній переклад / Остання помилка» у спливаючому вікні.

## Використання
1. Відкрийте спливаюче вікно → натисніть **Settings** (сторінка параметрів).
2. Налаштуйте базову URL-адресу API, ключ, модель, мови та режим виведення.
3. На будь-якій сторінці виділіть текст і скористайтеся контекстним меню або гарячою клавішею.
4. У X та коментарях YouTube натисніть «Перекласти текст» під підтримуваним блоком контенту, щоб побачити вбудований переклад. Ці кнопки можна вимкнути в розділі Settings.

## Налаштування (сторінка параметрів)
- Попередні налаштування постачальника та настроювана кінцева точка.
- Оновлення/завантаження моделі для OpenAI, Claude, Gemini, OpenRouter (користувацький/публічний, лише безкоштовний) і YandexGPT.
- Режим виведення + тривалість накладання.
- Перемикач кнопки швидкого вибору.
- Окремі перемикачі для показу кнопок перекладу в X і YouTube.
- Додаткова синхронізація ключа API між пристроями з попередженням безпеки.

## Дозволи
- Скрипт контенту працює на `<all_urls>`, щоб визначати виділення тексту й показувати вбудований інтерфейс перекладу там, де це підтримується.
- `contextMenus`, `storage`, `activeTab`, `scripting`, `notifications`
- Дозволи хосту для налаштованих постачальників API та API X/Twitter

## розвиток
- Завантажити розпаковане в `chrome://extensions`
- Файли записів: `background.js`, `content.js`, `popup.html`, `popup.js`, `options.html`, `options.js`

## Конфіденційність
Виділений текст надсилається лише до кінцевої точки API, яку ви налаштовуєте.
Перегляньте `PRIVACY_POLICY.md`.
