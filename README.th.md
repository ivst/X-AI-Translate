# AI แปลสำหรับ X และ YouTube

[![AI Translate in Chrome Web Store](ai-adv-large.png)](https://chromewebstore.google.com/detail/ai-translate-for-x-and-yo/ccgnhaicdhdhhangmfkddcippajhbbji)

Chrome Web Store: https://chromewebstore.google.com/detail/ai-translate-for-x-and-yo/ccgnhaicdhdhhangmfkddcippajhbbji

ส่วนขยาย Chrome สำหรับการแปลข้อความที่เลือก โพสต์ X (Twitter) และความคิดเห็นของ YouTube โดยใช้คีย์ API ของคุณเอง (ผู้ให้บริการเนทีฟที่เข้ากันได้กับ OpenAI และรองรับ)

## คุณสมบัติ
- แปลข้อความที่เลือกผ่านเมนูบริบทหรือปุ่มลัด `Alt+Shift+T` (เอาต์พุตสตรีมมิ่ง)
- ปุ่มแปลแบบอินไลน์ใต้โพสต์/ตอบกลับ X รายการ และความคิดเห็นของ YouTube (เอาต์พุตแบบสตรีม)
- ปุ่มแปลด่วนเสริมใกล้กับการเลือกข้อความ
- รองรับ OpenAI, Claude, Gemini, DeepSeek, YandexGPT, OpenRouter และปลายทางที่เข้ากันได้กับ OpenAI แบบกำหนดเอง
- รายการรุ่น OpenRouter พร้อมตัวกรองและแคชฟรีเท่านั้น
- รายการโมเดลการโหลดอัตโนมัติสำหรับ OpenAI, Claude, Gemini, OpenRouter และ YandexGPT
- โหมดเอาต์พุต: ขนมปังปิ้งด้านล่างขวาหรือโมดอลตรงกลาง (ไม่มีเอาต์พุตป๊อปอัปส่วนขยาย)
- เป้าหมาย + การเลือกภาษาต้นฉบับ (มีการตรวจจับอัตโนมัติ)
- คีย์ API สามารถจัดเก็บไว้ในเครื่อง (ค่าเริ่มต้น) หรือซิงโครไนซ์กับอุปกรณ์ต่างๆ (เป็นทางเลือกและมีความปลอดภัยน้อยกว่า)
- ผลลัพธ์บันทึกเป็น “การแปลล่าสุด / ข้อผิดพลาดครั้งล่าสุด” ในป๊อปอัป

## การใช้งาน
1. เปิดป๊อปอัป → คลิก **Settings** (หน้าตัวเลือก)
2. กำหนดค่า URL ฐาน API, คีย์, โมเดล, ภาษา และโหมดเอาต์พุต
3. ในหน้าใดก็ได้ ให้เลือกข้อความและใช้เมนูบริบทหรือปุ่มลัด
4. ใน X และคอมเมนต์ YouTube ให้คลิก "แปลข้อความ" ใต้เนื้อหาที่รองรับเพื่อดูการแปลแบบอินไลน์ และสามารถปิดปุ่มเหล่านี้ได้ใน Settings

## การตั้งค่า (หน้าตัวเลือก)
- ค่าที่ตั้งล่วงหน้าของผู้ให้บริการและจุดสิ้นสุดที่กำหนดเอง
- การรีเฟรชโมเดล/การโหลดสำหรับ OpenAI, Claude, Gemini, OpenRouter (ผู้ใช้/สาธารณะ ฟรีเท่านั้น) และ YandexGPT
- โหมดเอาต์พุต + ระยะเวลาการซ้อนทับ
- สลับปุ่มเลือกด่วน
- มีสวิตช์แยกสำหรับแสดงหรือซ่อนปุ่มแปลบน X และ YouTube
- การซิงค์คีย์ API เสริมระหว่างอุปกรณ์ต่างๆ พร้อมคำเตือนด้านความปลอดภัย

## สิทธิ์
- content script ทำงานบน `<all_urls>` เพื่อจับการเลือกข้อความและแสดง UI แปลแบบอินไลน์ในตำแหน่งที่รองรับ
- `contextMenus`, `storage`, `activeTab`, `scripting`, `notifications`
- สิทธิ์โฮสต์สำหรับผู้ให้บริการ API ที่กำหนดค่าและ X/Twitter API

## การพัฒนา
- โหลดที่แกะแล้วใน `chrome://extensions`
- ไฟล์รายการ: `background.js`, `content.js`, `popup.html`, `popup.js`, `options.html`, `options.js`
